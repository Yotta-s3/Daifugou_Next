# 大富豪 Web アプリ

React + Vite + TypeScript で構築した大富豪（大貧民）クライアントです。ホットシート（同一端末）でのプレイと簡易 CPU を両立しつつ、ゲームロジックは `src/game` に集約してオンライン化しやすい構造を意識しています。

## 開発

```bash
npm install
npm run dev
```

## 実装済み

- ロビー画面（開発ロードマップ + ゲーム開始導線）
- 設定画面（プレイヤー名 / 主要ルール / 7渡し・10捨て・Qボンバー / ジョーカー枚数切り替え / 設定のローカル保存）
- ホットシート + CPU 対戦
- ルール: しばり / 階段 / 革命 / 8切り / 11バック / 7渡し / 10捨て / Qボンバー / ジョーカー枚数変更
- 純関数型ゲームエンジン (`createGameState`, `applyAction`, `resolvePendingEffect` ほか) と CPU ロジック (`decideCpuAction`)
- HUDバッジで場の特殊状態・効果を常時表示
- React UI (カード選択、出す/パス、特殊効果選択、ログ、場ステータス、順位表示)

## 今後の拡張案

1. **設定プリセット & 難度選択** – 座席構成や CPU 難度、ルールセットをプリセット化し、ロビーから切り替え。
2. **オンライン化 (ルームID + WebSocket)** – サーバーで `GameState` を authoritative に扱い、クライアントはイベント送信のみに。Fastify/Next.js + ws を想定。
3. **昇格/降格ループ** – 複数ゲームを自動進行し、カード交換・階級入れ替え・スコアリングを適用。
4. **認証・ランキング** – ルーム管理、プレイヤープロフィール、戦績集計、観戦モードを追加。
5. **UI/UX 強化** – モバイル最適化、カードドラッグ操作、アニメーション、チャット/スタンプ、リプレイ機能。

サーバー実装に入る際は `src/game` 配下のロジックを共有パッケージ化し、クライアントとサーバーで同じ型/関数を利用する構成にすると安全です。
